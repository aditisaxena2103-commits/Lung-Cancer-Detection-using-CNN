# Lung Cancer Detection using CNN

## Project Overview
This project uses Convolutional Neural Networks (CNN) to detect lung cancer from chest X-ray images.

## Dataset
Dataset was taken from Kaggle containing labeled chest X-ray images.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- CNN

## Methodology
1. Image preprocessing
2. CNN model training
3. Model evaluation
4. Prediction on new X-ray images

## Results
The CNN model achieved approximately 85–90% accuracy on the test dataset.

## Applications
Early detection of lung cancer can assist doctors in diagnosis and treatment planning.
